## üìù HOW TO ADD IN GOOGLE ADS

### Step 1: Add Phrase Match Negatives (Lists 1, 2, 6)

1. Go to **Keywords** ‚Üí **Negative keywords**
2. Make sure you're at **Campaign level** (select "Avondale Property Management" campaign)
3. Click **+** (Add negative keywords)
4. Paste **List 1** (Competitor brands)
5. Select **Phrase match** from dropdown
6. Click **Save**
7. Repeat for **List 2** (Other cities) - Phrase match
8. Repeat for **List 6** (Tenant phrases) - Phrase match

### Step 2: Add Broad Match Negatives (Lists 3, 4, 5, 7)

1. Click **+** (Add negative keywords)
2. Paste **List 3** (Cities)
3. Select **Broad match** from dropdown
4. Click **Save**
5. Repeat for **List 4** (Competitor brands) - Broad match
6. Repeat for **List 5** (Tenant searches) - Broad match
7. Repeat for **List 7** (Wrong services) - Broad match
