## üí∞ HOW TO IMPLEMENT THE ROTATION

### Option 1: Manual Weekly Updates (Recommended for Start)
1. **Every Monday morning**, update campaign budgets in Google Ads
2. Set emphasis city to **$12/day**
3. Set other cities to minimal budgets ($0.25 - $2.00/day)
4. Keep a calendar reminder

### Option 2: Google Ads Scripts (Advanced)
- Automate budget changes with Google Ads scripts
- Script runs weekly and adjusts budgets automatically
- Requires technical setup but runs hands-free

### Option 3: Shared Budgets (Alternative)
- Create shared budgets for emphasis cities
- Manually switch which campaign uses the shared budget
- Less precise but easier to manage

## üîç MONITORING & OPTIMIZATION

### Track Weekly:
1. **Clicks per city** (should see spikes on emphasis weeks)
2. **Conversions per city** (which cities convert best?)
3. **CPC trends** (are they staying around $12?)
4. **Quality Score** (is it improving over time?)

### After 8 Weeks:
1. **Analyze which cities converted best**
2. **Adjust rotation** to give more weeks to best performers
3. **Consider increasing budget** for top 2-3 cities
4. **Reduce budget** for cities with zero conversions

### Optimization Rule:
- **If city converts during emphasis week:** Give it 2 weeks next cycle
- **If city has zero conversions after 2 emphasis weeks:** Reduce to minimal budget only
- **If city has 3+ conversions in emphasis week:** Consider making it permanent emphasis
