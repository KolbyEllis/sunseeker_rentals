# Avondale – Restore to Peak Settings (Jan 5–16 style)

When Avondale was performing at its peak (Jan 5–16), the campaign used **Manual CPC** and keyword-level bids, not Target CPA. Use these settings to restore that setup.

---

## 1. Bid strategy

- **Change from:** Maximize conversions (Target CPA $33.22)  
- **Change to:** **Manual CPC** (manual bidding)

**In Google Ads:**  
Campaign → **Settings** → **Budget and bidding** → set **Bidding** to **Manual CPC** (or “Manual bidding” / “Clicks”). Remove Target CPA.

---

## 2. Daily budget

- **Peak period:** ~$10–12/day  
- **Current:** $15/day  

You can keep **$15/day** (more room to get clicks) or set to **$12/day** to match peak. Either is fine.

---

## 3. Keyword-level bids (Manual CPC)

Set these **max CPC** bids at the keyword (or ad group) level so they match the peak setup:

| Keyword | Match type | Peak bid (restore to) | Note |
|--------|------------|------------------------|------|
| `[avondale property management]` | Exact | **$11.00** or **$12.50** | Winner keyword – main driver of conversions |
| `[property management avondale az]` | Exact | **$5.00** | Was $7 at peak; report said lower to $5 |
| `"avondale property management services"` | Phrase | **$7.00** | |
| `"avondale residential property management"` | Phrase | **$3.50** | (Google also suggested removing as redundant – optional to pause later) |
| Other exact/phrase keywords that had $2–3 | As before | **$2.00–$4.00** | Adjust only if you want to test higher |

**In Google Ads:**  
Campaign → **Ad groups** → select the ad group → **Keywords** → edit each keyword’s **Max. CPC** (or “Bid”) to the values above.

---

## 4. Optional (from peak report)

- **Pause “rarely served” keywords** that had 0 impressions (e.g. “avondale property management near me,” “best property management avondale,” etc.) so budget goes to the winners.
- **Pause** `"avondale property management company"` if it still has impressions but 0 clicks (low relevance).

---

## Summary checklist

1. Set bidding to **Manual CPC** (remove Target CPA).  
2. Keep daily budget at **$12** or **$15**.  
3. Set **keyword bids** as in the table above (winner at $11–12.50, others as listed).  
4. Save and let run 5–7 days; then check impressions/clicks/conversions.

After you have 15–30+ conversions again, you can consider switching back to **Maximize conversions** (with a realistic Target CPA) if you want automated optimization.
