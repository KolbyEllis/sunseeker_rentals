## üéØ ROOT CAUSE ANALYSIS

### **Why This Is Happening:**

1. **Performance Max Campaign Type:**
   - Performance Max is Google's fully automated campaign type
   - It automatically targets across Search, Display, YouTube, Gmail, Discover, and Maps
   - **Problem:** It's optimizing for clicks/engagement, not conversions
   - **Problem:** It has too much freedom to show ads on irrelevant searches

2. **Lack of Negative Keywords:**
   - No negative keywords visible in the report
   - Google is matching to any search that might be "relevant"
   - Terms like "houses for rent", "apartments", "studios" should be negative keywords

3. **No Conversion Tracking:**
   - 0 conversions suggests conversion tracking isn't set up
   - Performance Max needs conversion data to optimize properly
   - Without conversions, it optimizes for clicks (which is why you're getting tenant searches)

4. **Broad Targeting:**
   - Performance Max is casting too wide a net
   - It's showing ads to anyone searching property/rental-related terms
   - Not filtering for property owners vs. tenants
